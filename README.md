## サービス概要
日常のお散歩や外出を「冒険のクエスト」へと変えてしまう、ミッション生成型の散策エンターテインメントアプリです。
位置情報やランダム要素を組み合わせて「指定された座標にある橋を渡れ」や「10分以内に青い看板を3個探せ」といったミッションを提示し、現実の世界を探索しながらクリアを目指します。
ミッション達成を通じてストーリーが進展したり、相棒となるキャラクターが進化・成長したりする育成要素を盛り込み、ただの移動をワクワクするゲーム体験へと進化させます。

## このサービスへの思い・作りたい理由
【エピソード】
最近、友人とダイエットのために散歩を始めようという話になりました。しかし、私自身これまで散歩の習慣がほとんどなく、「目的地もなくただ歩き続ける」という行為にどうしても楽しさを見出せず、もっと散歩そのものをワクワクする体験に変えられないかと考えたのが始まりです。
また、近年の若い世代において、糖尿病治療薬をダイエット目的で不適切に使用する「美容瘦せ」が流行し、本来の患者に薬が行き届かないという深刻な社会問題も耳にしました。薬などの安易な手段に頼るのではなく、自分の足で歩く喜びを知り、健康的な体づくりを支援したいという願いも込めています。

【元となる思い】
「痩せなければならない」という義務感を、「冒険していたら結果的に健康になっていた」というポジティブな体験へ変換したいという思いが根底にあります。
特に大切にしたいのは、アプリのミッションを通じて、これまで素通りしていた近所の何気ない景色や、四季の移ろいに目を向けるきっかけを作ることです。「あそこにこんな花が咲いていたんだ」「この看板、面白い形をしているな」といった小さな発見を積み重ねることで、日常の風景が色鮮やかな冒険の舞台に変わります。キャラクターの成長と現実世界の探索をリンクさせることで、心身ともに健やかになれる「歩く理由」を提供したいと考えています。
