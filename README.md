## サービス概要
日常のお散歩や外出を「冒険」へと変えてしまう、ミッション生成型の散策エンターテインメントアプリです。
位置情報やランダム要素を組み合わせて「指定された座標にある橋を渡れ」や「10分以内に青い看板を3個探せ」といったミッションを提示し、現実の世界を探索しながらクリアを目指します。
ミッション達成を通じてストーリーが進展したり、相棒となるキャラクターが進化・成長したりする育成要素を盛り込み、ただの移動をワクワクするゲーム体験へと進化させます。

## このサービスへの思い・作りたい理由
【エピソード】
最近、友人とダイエットのために散歩を始めようという話になりました。しかし、私自身これまで散歩の習慣がほとんどなく、「目的地もなくただ歩き続ける」という行為にどうしても楽しさを見出せず、もっと散歩そのものをワクワクする体験に変えられないかと考えたのが始まりです。
また、近年の若い世代において、糖尿病治療薬をダイエット目的で不適切に使用する「美容瘦せ」が流行し、本来の患者に薬が行き届かないという深刻な社会問題も耳にしました。薬などの安易な手段に頼るのではなく、自分の足で歩く喜びを知り、健康的な体づくりを支援したいという願いも込めています。

【元となる思い】
「痩せなければならない」という義務感を、「冒険していたら結果的に健康になっていた」というポジティブな体験へ変換したいという思いが根底にあります。
特に大切にしたいのは、アプリのミッションを通じて、これまで素通りしていた近所の何気ない景色や、四季の移ろいに目を向けるきっかけを作ることです。「あそこにこんな花が咲いていたんだ」「この看板、面白い形をしているな」といった小さな発見を積み重ねることで、日常の風景が色鮮やかな冒険の舞台に変わります。キャラクターの成長と現実世界の探索をリンクさせることで、心身ともに健やかになれる「歩く理由」を提供したいと考えています。

## ユーザー層について
1. 写真や動画で「自分だけの発見」を表現したいSNSユーザー（Instagram/TikTok）
【理由】
どこにでもある流行りのカフェではなく、誰も知らない「路地裏の面白い看板」や「季節の小さな変化」を、自分なりの視点で切り取りたい欲求を持つ層です。ミッションというきっかけがあれば、彼らは能動的に街を歩き、質の高い投稿を量産してくれるため、アプリの魅力を外部へ強力に発信してくれるエンジンになると考えたからです。

2. 運動への苦手意識が強い、インドア派のインドア派のゲーム・キャラクター愛好家（オタク層）
【理由】
「健康のための運動」には興味がなくても、「キャラの育成」や「ストーリーの解放」といった目的があれば動ける、ポテンシャルの高い運動不足層です。彼らに「キャラを育てるために外に出たら、意外と景色も良くて楽しかった」という成功体験を提供することで、運動を苦行ではなく「自分の相棒との冒険の時間」へと再定義し、健康への意識を根底から変えてもらいたいからです。

## サービスの利用イメージ
■ ストーリー背景
本サービスは、人々に忘れられ力を失った「小さな土地神」の再生をストーリーとして進んでいきます。姿と記憶を失い、色が抜け落ちて「白紙」のような状態になった神様。ユーザーは、日常の風景を「捧げもの」として届け、共に歩くことで神様との絆を育み、そのエネルギーによって神様の姿と心を取り戻していく育成散策アプリです。

■ 遊び方のサイクル
1．冒険の開始
アプリを起動し、まだ姿のぼんやりした「白紙状態の神様」と挨拶をして散歩に出発します。スマホの歩数計と連動し、一緒に歩く時間がそのまま神様との共有体験になります。

2．探索とお題のクリア
散歩中、「季節の花を撮れ」「橋を渡れ」といったお題（ミッション）が届きます。これらをクリアしたり、道中の景色を「捧げもの」として写真に撮ったりすることで、神様との**「絆（親密度）」**がリアルタイムで深まっていきます。

3．エネルギーの蓄積と進化
一緒に歩いた距離や、捧げられた景色のエネルギーが一定量貯まると、神様のビジュアルが変化します。白紙だった体に鮮やかな色が宿り、人型へと「進化」を遂げます。

4．継続的な利用動機の創出
「共に歩んで絆を深め、その結果として相棒が進化する」というプロセスが、ユーザーに強い愛着を生みます。絆が深まるほど、神様の表情が豊かになったり、画面上でのアクションが変化したりと、目に見える反応が変化します。「自分の歩みが相棒の成長に直結する」という手応えが、毎日の外出を楽しみなイベントへと変え、利用動機を創出します。

## ユーザーの獲得について
・ハッシュタグによる大喜利文化の創出
アプリ内のミッションで見つけた「変な看板」「二度見してしまう景色」「季節の珍しい風景」などを、ハッシュタグ「#散歩で見つけた異変」と共にSNSへ投稿する流れを作ります。「ただの散歩」を「ネタ探しの冒険」に変換し、大喜利のような感覚で面白がるコミュニティを可視化することで、非ユーザーに対しても「このアプリがあれば、近所を歩くだけでこんなに面白い発見があるんだ」というワクワク感を伝播させます。

・「推し活×散歩」によるオタクユーザーの獲得
「ぬい撮り」などの趣味を持つ層に対し、散歩ミッションを通じて**「自分の大切なものを最高に輝かせる、自分だけのロケーションを探し出す体験」**を提供します。「今の時期しか咲いていない花を探せ」「表情のあるレトロな壁を見つけろ」といった、視点を変えるお題をきっかけに、ユーザーを自発的な「ロケハン散歩」へと誘います。撮影した写真は、場所や日付が記録された洗練されたデザインフレームで保存でき、「#ぬい散歩」**という独自のハッシュタグと共にSNSへ投稿する流れを作ります。これにより、熱量の高いコミュニティ内での連鎖的な認知拡大を狙います。

## サービスの差別化ポイント・推しポイント
他アプリ・サービスとの差別化ポイント
似たような外出を自発的に促すゲーム『Pokémon GO』や『ピクミン ブルーム』との対照的な設計
既存の主要アプリは、あらかじめ設定された「ポケストップ」や「ポスト」という**特定の目的地（点）へユーザーを誘導します。対して本サービスは、道端の「変な看板」や「季節の彩り」といった、システムが把握しきれない道中の些細な発見（プロセス）**を遊びの主役にする点が最大の差別化です。運営に「歩かされる」のではなく、自分の感性で「歩く理由を見つける」能動的な体験を提供します。

推しポイント
「散歩」という行為自体が、中毒性のある娯楽に変わる体験
ダイエットや歩数稼ぎといった「将来の健康」のためではなく、今この瞬間の「面白いものを見つけたい」「自分だけの景色を切り取りたい」という好奇心を原動力にします。努力感や義務感を一切排除し、無心で街を探索しているうちに「気づけば数キロ歩いていた」という没入感こそが一番の推しです。

## 機能候補
1．MVP
・ランダムお題（ミッション）生成機能
「季節の色」「橋を渡れ」「10分間歩け」など、場所や状況に応じたお題を提示。
・撮影・収集インターフェース
お題をクリアした際や景色を撮った際、それがキャラクターに吸収されるような演出。
・キャラクター・進化システム（第一段階）
歩数やミッション達成度を「エネルギー」として蓄積。
・絆（親密度）メーター
現在の絆の深さを可視化する簡易ステータス画面。
・簡易歩数計・ログ・マップ表示

2．本リリース
・スポット共有マップ（匿名制）
他のユーザーが発見した「異変」の場所を共有。安全のため地点は曖昧に表示。
・思い出の記憶（ギャラリー機能）
絆が深まった際に解放される「街の思い出イラスト」の閲覧図鑑。
・キャラクター・リアクション拡張
絆レベルに応じた「挨拶」や「アクション（動き）」の変化。

## 使用する技術スタック
- 使用するフレームワーク：Ruby 3.2.2 / Ruby on Rails 7.1.3
- データベース：PostgreSQL
- デプロイ先：Render
- 使用予定のライブラリ：
Google Maps JavaScript API: 高精度なマップ表示および地点情報の取得。
Tailwind CSS: スマホの片手操作に特化したUI（ボタン配置など）を組むため。
geocoder (Gem): 緯度・経度の取得や距離計算を正確に行うため。
devise (Gem): ユーザー認証およびセキュリティ機能を適切に実装するため。
